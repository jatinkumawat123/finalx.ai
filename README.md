# FinalX.AI — Setup Guide

## Vercel par FREE Deploy karne ke steps:

### Step 1 — GitHub Account banao
👉 github.com par free account banao

### Step 2 — Yeh folder upload karo
1. github.com/new par jao
2. Repository name: "finalx-ai"
3. "uploading an existing file" click karo
4. Is folder ke saare files drag & drop karo
5. "Commit changes" click karo

### Step 3 — Vercel par deploy karo
1. vercel.com par jao (free account banao)
2. "Add New Project" click karo
3. GitHub se apna "finalx-ai" repo select karo
4. "Deploy" click karo

### Step 4 — API Key lagao (IMPORTANT)
1. Vercel dashboard mein apna project open karo
2. "Settings" → "Environment Variables" jao
3. Name: ANTHROPIC_API_KEY
4. Value: Anthropic ka API key (console.anthropic.com se lo — free credit milta hai)
5. "Save" karo
6. "Redeploy" karo

### Step 5 — Live!
Tumhara website live hoga: https://finalx-ai.vercel.app

## Anthropic API Key kaise milega?
1. console.anthropic.com par jao
2. Free account banao
3. "API Keys" → "Create Key"
4. Key copy karo → Vercel mein paste karo
